git&github学习总结

git&github学习总结1.Git的优势 大部分操作在本地完成，不需要联网 完整性保证 尽可能添加数据而不是删除或修改数据 分支操作非常快捷流畅 与 Linux 命令全面兼容 2.Git的基本操作 2.1 状态查看 git stastus 查看工作区、暂存区的状态 2.2 添加 git add [file name] 将工作区的“新建/修改”添加到暂存区 2.3 提交 git commit -m "commit message" [file name] 将暂存区的内容提交到本地库 2.4 查看历史记录 git log 多屏显示控制方式： 空格向下翻页 b 向上翻页 q 退出 2.5 前进后退 本质：HEAD指针移动 基于索引值操作[推荐] git reset --hard [局部索引值] git reset --hard a6ace91 使用^符号：只能后退 git reset --hard HEAD^ 注：一个^表示后退一步，n个表示后退n步使用~符号：只能后退 git reset --hard HEAD~
