if__name__=='__main__'的使用

一个 Python 源码文件除了可以被直接运行外，还可以作为模块（也就是库）被导入。 if__name__=='__main__'的作用就是控制这两种情况执行代码的过程，在if __name__ == '__main__': 下的代码只有在第一种情况下（即文件作为脚本直接执行）才会被执行，而import到其他脚本中是不会被执行的。 name是当前模块名，当模块被直接运行时模块名为_main_，也就是当前的模块，当模块被导入时，模块名就不是main，即代码将不会执行。 每个python模块都包含内置的变量 name，当该模块被直接执行的时候，name 等于文件名（包含后缀 .py ）；如果该模块 import 到其他模块中，则该模块的 name 等于模块名称（不包含后缀.py）。 而 “main” 始终指当前执行模块的名称（包含后缀.py）。进而当模块被直接执行时，name == ‘main’ 结果为真;而当模块被import到其他模块中时，name == ‘main‘结果为假，就是不调用对应的方法。
